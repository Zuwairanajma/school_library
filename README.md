# 📖 [Oop-school-library] <a name="about-project"></a>

**[Oop-school-library]**  Is an app that a Librarian at a school university uses to create new user (students/teachers), add new books, and save records of who borrowed a given book and when.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

 - `bundle install`

### Setup

Clone this repository to your desired folder:

- cd `school-library`
- `git clone https://github.com/Zuwairanajme/school-library.git`


### Install

Install this project with:

- cd Oop-school-library
- bundle install

### Run tests

To run tests, run the following command:

- `rubocop --color`
